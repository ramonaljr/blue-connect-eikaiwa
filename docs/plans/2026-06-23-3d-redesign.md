# 3D / WebGL Redesign Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Redesign Landing, Login/Signup, Dashboard home, and all 10 dashboard modules with real WebGL 3D (Three.js / react-three-fiber) and bold, immersive motion, behind a single global canvas with capability-tiered fallbacks.

**Architecture:** One global `<Canvas>` mounted in the app shell; every surface declares lightweight drei `<View>` regions tracked to DOM elements (Option A). A `useDepthCapability()` hook gates `full` / `lite` / `off` tiers; `off` (no WebGL or `prefers-reduced-motion`) falls back to the existing `.bg-gradient-mesh` + framer-motion. Server components and data-fetching are untouched — all 3D lives in client `*-content.tsx` components.

**Tech Stack:** Next.js 16 (App Router), React 19, TypeScript, Tailwind v4 (OKLCH), framer-motion v12, `three`, `@react-three/fiber` v9, `@react-three/drei`, `@react-three/postprocessing`, `maath`, Vitest.

**Reference:** `docs/plans/2026-06-23-3d-redesign-design.md`

**Step 3: Implement** — `src/lib/three/capability.ts`:
```ts
export type DepthTier = 'full' | 'lite' | 'off'

export interface CapabilitySignals {
  webgl2: boolean
  reducedMotion: boolean
  cores: number
  memory: number // GB; 0 when unknown
  coarse: boolean // coarse pointer (touch)
}

export function resolveTier(s: CapabilitySignals): DepthTier {
  if (!s.webgl2 || s.reducedMotion) return 'off'
  const lowCores = s.cores > 0 && s.cores <= 4
  const lowMemory = s.memory > 0 && s.memory <= 3
  if (lowCores || lowMemory || s.coarse) return 'lite'
  return 'full'
}
```

**Step 4: Run test** → PASS.

**Step 5:** Implement the hook — `src/lib/three/use-depth-capability.ts`:
```ts
'use client'
import { useEffect, useState } from 'react'
import { resolveTier, type DepthTier } from './capability'

function detectWebGL2(): boolean {
  try {
    const c = document.createElement('canvas')
    return !!c.getContext('webgl2')
  } catch {
    return false
  }
}

export function useDepthCapability(): DepthTier {
  // SSR-safe default: 'off' so nothing 3D renders until we confirm capability.
  const [tier, setTier] = useState<DepthTier>('off')

  useEffect(() => {
    const reducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches
    const coarse = window.matchMedia('(pointer: coarse)').matches
    const cores = navigator.hardwareConcurrency ?? 0
    // deviceMemory is non-standard; guard it.
    const memory = (navigator as Navigator & { deviceMemory?: number }).deviceMemory ?? 0
    setTier(resolveTier({ webgl2: detectWebGL2(), reducedMotion, cores, memory, coarse }))

    const mq = window.matchMedia('(prefers-reduced-motion: reduce)')
    const onChange = () =>
      setTier(resolveTier({ webgl2: detectWebGL2(), reducedMotion: mq.matches, coarse, cores, memory }))
    mq.addEventListener('change', onChange)
    return () => mq.removeEventListener('change', onChange)
  }, [])

  return tier
}
```
