# Fully Featured Learner Dashboard Modules — Design Document

**Date:** 2026-03-19
**Status:** Approved

## Overview

Complete production-depth implementation of all 8 learner dashboard modules: Dashboard, AI Chat, AI Voice, Courses, Lessons, Tutors, Progress, and Settings. Delivered in 5 feature slices ordered by dependency chain.

## Implementation Strategy: Feature Slices

| Slice | Modules | Rationale |
|-------|---------|-----------|
| 1. Foundation | Settings + Dashboard | Establishes user identity, preferences, and the personalized hub |
| 2. AI Learning | AI Chat + AI Voice + Voice Server | Core differentiator — AI-powered conversation practice |
| 3. Structured Learning | Courses + Exercises + Adaptive Difficulty | Self-paced learning with full exercise suite |
| 4. Live Learning | Tutors + Lessons + Booking + Daily.co | Human tutor marketplace and lesson management |
| 5. Gamification | Progress + Achievements + XP/Streak | Wraps all modules with engagement and analytics |

### AI Voice Module

#### Three Sub-Modes (Tab Navigation)

**1. Voice Chat (Guided)**
- Same scenario picker as text chat
- Main UI: large waveform visualizer showing when AI is speaking vs listening
- Live transcript panel on the side showing both parties' speech in real-time
- Pronunciation score per utterance displayed inline (color-coded: green >80, yellow 60-80, red <60)
- "Slow down" button → AI speaks slower on next response
- Session timer with max duration based on subscription tier
- End session → summary screen

**2. Immersive Roleplay**
- Pre-built scenarios with richer context (character descriptions, situation setup)
- No transcript visible by default (toggle to show)
- More natural conversation flow — AI stays in character
- Scenario completion detection (AI recognizes when the roleplay objective is met)
- End session → detailed report: scenario completed/not, key phrases used, errors, pronunciation highlights

**3. Pronunciation Practice**
- Targeted drills: AI says a word/phrase → learner repeats → Azure scores it
- Focus areas: minimal pairs (light/right, think/sink), word stress, intonation patterns
- Visual feedback: phoneme-level breakdown showing which sounds were off
- Spaced repetition: words you struggle with come back more frequently
- Progress tracking per sound category

#### Voice Session Summary Screen (All Modes)
- Overall pronunciation score (average)
- Duration
- Corrections made (grammar/vocabulary)
- Words/phrases to review (auto-saved to phrase bank)
- "Practice again" / "Try different scenario" buttons
- Score saved to `ai_conversations` table

#### Voice WebSocket Server (Railway)
```
Browser (MediaRecorder)
  → WebSocket → Railway Server
  → Deepgram STT (streaming)
  → Claude API (streaming response)
  → ElevenLabs TTS (streaming audio)
  → WebSocket → Browser (AudioContext playback)

  Parallel: Azure Speech SDK → pronunciation assessment → WebSocket → Browser
```

- Server maintains session state: conversation history, scenario context, user preferences
- Handles graceful disconnects and reconnection
- Auth via JWT token passed on WebSocket handshake
- Session recording stored to Supabase Storage (opt-in)

#### Voice Server Structure
```
voice-server/
  src/
    index.ts          — WebSocket server entry
    session.ts        — Session state management
    deepgram.ts       — STT streaming client
    claude.ts         — Claude API conversation
    elevenlabs.ts     — TTS streaming client
    azure-speech.ts   — Pronunciation assessment
    auth.ts           — JWT verification
  package.json
  Dockerfile
```

### Course Detail Page

#### Overview Tab
- Course title, description (bilingual), level, category
- Learning objectives list
- Unit breakdown with progress indicators per unit (locked/available/in-progress/completed)
- Overall course progress bar with percentage
- Estimated remaining time
- "Continue" / "Start next unit" primary CTA

#### Units Navigation
- Accordion or vertical stepper showing all units
- Each unit shows: title, exercise count, estimated time, status icon
- Linear progression: units unlock sequentially
- Final assessment unit at the end

### Unit Detail Page (New)

#### Content Area
- Rich content rendered from jsonb: text blocks, images, audio clips, vocabulary tables, grammar explanations
- Bilingual toggle: show English only / show Japanese translation alongside
- Vocabulary words are tappable → shows definition, pronunciation (audio), example sentence
- "Add to phrase bank" on any vocabulary item

#### Exercise Flow
- Exercises appear after content, in sequence
- Progress bar across top showing current exercise / total

### Exercise Types (Full Implementation)

1. **Multiple Choice** — Enhanced with hints (deducts points), retry option, bilingual explanation
2. **Fill in the Blank** — Multiple blanks, word bank option vs free text, case-insensitive
3. **Matching** — Drag-and-drop pairs (word↔meaning, sentence halves, image↔word), timed option
4. **Reorder** — Drag to arrange words/sentences, mobile-friendly touch support
5. **Free Response** — AI-graded by Claude with rubric, returns score + corrected version + bilingual explanation
6. **Audio Exercise (New)** — Listen and respond: comprehension MC, dictation, pronunciation repeat
7. **Conversation Exercise (New)** — Mini 3-5 exchange AI chat using target vocabulary/grammar

### Adaptive Difficulty System

- Track accuracy per exercise type and skill area (grammar, vocabulary, listening, pronunciation)
- Rolling accuracy over last 20 exercises per skill
- Accuracy > 85%: harder exercises (more free response, less MC, faster timers)
- Accuracy < 60%: easier exercises (more hints, simpler vocab, more matching/MC)
- Stored in `skill_profiles` table

### Final Assessment
- Last unit in course, timed (30 min for 20 questions)
- Mix of all exercise types weighted by course content
- 70%+ required to complete course
- Retake after 24-hour cooldown
- Results: score breakdown by skill area, time taken, comparison to average

## Slice 4: Live Learning — Tutors + Lessons + Booking + Daily.co

### Tutors Module

#### Search-First Path (Tutors List Page Enhanced)
- Filter bar: date/time picker, specialty, tutor type (certified/community/all), price range slider, minimum rating, language spoken
- Sort by: recommended, rating, price, most experienced
- "Available now" toggle — shows tutors with open slots in next 2 hours
- Real-time filter updates

#### Tutor Profile Page (New: `/dashboard/tutors/[id]`)

**Header Section**
- Large avatar, display name, certified badge, star rating + lesson count + review count
- "Book a Lesson" primary CTA (sticky on mobile)
- Online/offline status indicator

**About Tab**
- Bio (bilingual), languages spoken, specialties, teaching style, video introduction (optional)

**Schedule Tab (Calendar-First Path)**
- Weekly calendar view (respects learner's timezone)
- Available slots as green blocks, booked slots greyed out
- Click slot → booking confirmation modal
- Toggle 25min / 50min duration
- Navigate 4 weeks ahead

**Reviews Tab**
- Review list with avatar, rating, text, date
- Category breakdown: communication, patience, expertise, value
- Pagination (10 per page)

#### Booking Flow
1. Select slot → booking modal (tutor, date/time in learner TZ + JST, duration, credit cost)
2. Credit check → confirm or redirect to purchase
3. On confirm: create lesson, deduct credit (earliest expiring first), notify both parties, schedule reminders (24hr + 1hr)
4. Confirmation with "Add to Calendar" (.ics download)

### Lessons Module

#### Lessons List Page (New: `/dashboard/lessons`)

**Tabs: Upcoming / Past / Canceled**

**Upcoming:** Cards with tutor, date/time + countdown, duration, actions (Prepare, Join, Reschedule, Cancel with policy enforcement)

**Past:** Cards with tutor, date, rating, actions (Review, View Notes, Rebook, Download Notes as PDF)

#### Cancellation Policy Enforcement
- 24+ hrs: full refund
- 2-24 hrs: 50% refund
- < 2 hrs: no refund
- Confirmation required for all

#### Lesson Preparation Area
- "Topics I want to cover" — free text
- "Vocabulary I want to practice" — tag input
- "Goals for this lesson" — checkboxes
- Shared with tutor via their dashboard

#### Lesson Room (Enhanced)
- Pre-join: camera/mic preview, device selector, lesson details, prep notes
- In-session: Daily.co video + sidebar tabs (Chat, Notes, Phrases) + lesson timer + recording toggle
- Chat and Notes are real-time via Supabase Realtime

#### Post-Lesson Flow
1. Recording upload to Supabase Storage, AI summary via Claude
2. Rating (1-5) with category breakdown + written review
3. "What did you learn?" reflection prompt
4. 100 XP awarded

#### Review System
- Category breakdown stored as jsonb
- Tutor average_rating updated via DB trigger
- Flagging: below 3.5 (10+ reviews) → admin review; certified below 4.0 → warning

#### Activity Tab
- Chronological feed of all learning activities with filters
- Weekly summary card with comparison to previous week

#### Achievements Tab

**Categories:**

| Category | Achievements |
|----------|-------------|
| Consistency | 7/30/100/365-day streak |
| AI Practice | First chat, 10/100 chats, First voice, 50 voice sessions |
| Pronunciation | Score 80+/90+/95+, Master a sound category |
| Courses | First/5/all-in-category completed |
| Lessons | First/10/50 lessons, Leave a review |
| Social | First tutor review, 100 saved phrases |
| XP | Level 5/10/25/50 |
| CEFR | Reach A2/B1/B2/C1/C2 |

- Grid display: icon, title (bilingual), description, progress bar, unlocked date
- Locked as greyed silhouettes with requirements visible
- Unlock animation: confetti + toast
- Shareable achievement cards (generated image)

#### Goals Tab
- Weekly goals: learner sets targets, progress bars, auto-suggested goals
- Goal completion → 50 XP bonus
- Monthly platform challenge with leaderboard position

#### Leaderboard (Opt-in)
- Weekly XP, resets Monday JST
- Top 50 + your position
- Anonymous display names option
- Separate boards: overall, AI practice, courses

### XP System (Cross-Module)

| Activity | XP |
|----------|-----|
| Exercise completed | 10-50 (by difficulty) |
| AI text chat (5+ messages) | 30 |
| AI voice session (3+ min) | 50 |
| Pronunciation drill (10+ attempts) | 30 |
| Live lesson completed | 100 |
| Course unit completed | 75 |
| Course completed | 200 |
| Daily goal met | 50 |
| Weekly goal met | 100 |
| Achievement unlocked | 25-500 |

Server action `awardXP(userId, amount, source)` called from each module.

### Streak System
- Any learning activity counts toward daily activity
- Increments if `last_activity_date` was yesterday, resets if gap > 1 day
- Streak freeze: Premium users get 2/month, deducts from `streak_freezes_remaining`
- Updated via `updateStreak()` server action

### Weekly Summary Email
- Sent Monday 9:00 AM JST via Resend, in Japanese
- Stats, streak, achievements, recommendations
- Unsubscribable via Settings
